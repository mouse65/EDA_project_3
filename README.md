# 🏨 EDA и подготовка признаков для прогнозирования рейтингов отелей (Kaggle Booking Reviews)
Этот проект выполнен в рамках соревнования [SF‑DST] Booking Reviews на Kaggle.
Цель — исследовать данные отзывов об отелях, выявить закономерности, подготовить качественные признаки и построить модель для прогнозирования рейтинга.

## 🎯 Цели проекта
- Провести разведочный анализ данных (EDA).
- Найти важные закономерности в данных о бронированиях и отзывах.
- Создать новые числовые признаки.
- Оценить значимость признаков.
- Удалить нерелевантные или шумовые фичи.
- Построить модель и получить метрику MAPE.
